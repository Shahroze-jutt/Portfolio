Lead Qualification Agent (Olivia)

Status: Production-ready Timeline: 10-node flow Impact: Days → Minutes

Problem

Sales teams spent days manually qualifying leads after form submissions. By the time they called, leads had already engaged with competitors.

Solution Architecture

Form Trigger → Phone Format → Retell AI Call
↓
Airtable Create → Dynamic Variables → Qualification Script
↓
MEDDIC-Lite Questions → Scoring → Auto-Book (if qualified)
↓
Calendar Sync → Email Notification → CRM Update

Key Features

  • MEDDIC-Lite qualification: Match, Pain, Budget, Authority, Timeline, Intent
  • Dynamic personalization: Calls leads by first name, references desired outcome
  • Auto-booking: High-quality leads directly book meetings (no manual follow-up)
  • E.164 phone formatting: Handles international numbers (Pakistan +92 format)
  • Objection handling: Scripted responses for common rejection patterns

Results
